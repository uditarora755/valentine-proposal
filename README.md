# Valentine Proposal Page

A sweet, playful Valentine page with Dudu & Bubu gifs and a mischievous “No” button for mobile.

## Run locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## Make it live

### Option 1: GitHub Pages (free)

1. Create a GitHub repo and push this folder.
2. In **Settings → Pages**, set **Branch** to `main` and **/root`**.
3. Save, and GitHub will give you a public URL in a few seconds.

### Option 2: Netlify Drop (fastest)

1. Go to <https://app.netlify.com/drop>.
2. Drag and drop the folder with `index.html`.
3. Netlify instantly gives you a public link.

### Option 3: Vercel (simple)

1. Install Vercel: `npm i -g vercel`.
2. Run `vercel` in the project folder and follow the prompts.
3. You get a public URL right away.
